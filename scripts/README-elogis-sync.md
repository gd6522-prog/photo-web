# elogis 점포마스터 반자동 동기화

실행:

```bash
npm run sync:store-master
```

동작 순서:

1. Chrome 전용 자동화 창이 열린다.
2. elogis 로그인이 안 되어 있으면 사용자가 직접 로그인한다.
3. 엔터를 누르면 스크립트가 이어서 진행한다.
4. `TMS 시스템 로그인`으로 들어간다.
5. `계획관리 > 노선-점포(배송처)매핑`으로 이동한다.
6. 배송그룹 `상온`을 선택하고 조회한다.
7. 엑셀을 다운로드한다.
8. 엑셀에서 `호차번호, 배송순서, 배송처코드, 배송처명`을 추출한다.
9. `store_map` 테이블에 자동 반영한다.

생성 경로:

- 전용 브라우저 프로필: `.automation/elogis-profile`
- 다운로드 파일: `.automation/downloads`
- 정규화 결과 JSON: `.automation/output`

주의:

- 고객사 내부망 사이트는 다운로드만 수행한다.
- 삭제/수정 같은 위험 동작은 하지 않는다.
- 로그인 세션 유지를 위해 브라우저는 자동 종료하지 않는다.
- UI 구조가 바뀌면 클릭 selector를 다시 맞춰야 할 수 있다.
